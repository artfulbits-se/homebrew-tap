# Homebrew Tap for e-bash

This is the official Homebrew tap for [e-bash](https://github.com/OleksandrKucherenko/e-bash).

## Installation

```bash
brew tap artfulbits-se/tap https://github.com/artfulbits-se/homebrew-tap.git
brew install e-bash
```
